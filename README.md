# BeeCrowd Solutions

A collection of accepted solutions to [BeeCrowd](https://www.beecrowd.com.br/) (formerly URI Online Judge) problems, written in Python (and SQL).

## Structure

Solutions are organized by category under `accepted/`:

```
accepted/
├── 1_beginner/      # Basic I/O, arithmetic, simple logic
├── 2_ad-hoc/        # Problem-specific logic, simulations
├── 3_strings/       # String manipulation and parsing
├── 4_structures/    # Stacks, queues, trees, hash tables
├── 5_math/          # Number theory, combinatorics
├── 6_paradigms/     # Recursion, DP, divide & conquer
├── 7_graphs/        # Graph traversal, shortest path, trees
└── 9_sql/           # Database queries
```

Files follow the naming convention:
```
BEE-{problem_id}-{category}-{difficulty}_{Problem-Title}.py
```
